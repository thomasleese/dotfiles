# dotfiles

Here I store my dotfiles configuration.

## Installation

```sh
$ brew install gh
$ gh auth login
$ cd ~/Developer
$ gh repo clone dotfiles
$ cd dotfiles
$ ./install.sh
```

## Symlinks

```
~/.config/fish/conf.d -> fish/conf.d
~/.gitignore_global -> git/gitignore
~/.atom/config.cson -> atom/config.cson
```
